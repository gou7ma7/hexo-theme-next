Framwork [Hexo](https://hexo.io/);

Theme [Hexo](https://theme-next.js.org/);

## Quick Start

### Create a new post

``` bash
$ hexo new "my_new_post"
```

my_new_post: 推荐使用下划线分割，否则站内跳转文章链接的时候markdown可能会出错

### Generate static files

``` bash
$ hexo generate
```
和下文中-g 效果一致

### Run server

``` bash
$ hexo server -g
```

### Deploy to remote sites

``` bash
$ hexo deploy -g
```
